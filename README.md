Setup
-----
0) Trouble figuring out what calendar week it is? Then, this is the app for you.
1) .NET6.0 runtime is a pre-requirement. (If required, please install the included 'windowsdesktop-runtime-6.0.3-win-x64.exe')
2) Launch the 'WhatCalendarWeek.exe' application.
3) Enjoy!

Concept
-------
1) Application shows current date and 'Calendar week' number on startup.
2) Select a desired date on the calendar and the 'Calendar week' number gets generated.
3) Type in a desired 'Calendar week' number and click the '^' button, the date for the corresponding week is highlighted in the calendar.
4) Reset to today's date and Calendar week by clicking the 'Today' button.
